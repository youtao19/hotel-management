## 🎯 业务逻辑说明

### 核心规则
1. **营业日定义**: 从每天 8:00 开始，到次日 8:00 结束
2. **交接对象**: 交接的是"前一个营业日"的工作和账目
3. **保存日期**: 保存的是"要交接的营业日"的日期
4. **查询记录**: 查询的是"要交接的营业日的前一天"的交接记录

### 日期计算公式
```javascript
// 当前营业日
currentBusinessDate = (当前时间 < 8:00) ? 昨天 : 今天

// 要交接的营业日
handoverBusinessDate = currentBusinessDate - 1天

// 要查询的记录
queryDate = handoverBusinessDate - 1天

// 要保存的日期
saveDate = handoverBusinessDate
```

## 📝 关键代码片段

### formatLocalDate 函数
```javascript
const formatLocalDate = (date) => {
  const year = date.getFullYear()
  const month = String(date.getMonth() + 1).padStart(2, '0')
  const day = String(date.getDate()).padStart(2, '0')
  return `${year}-${month}-${day}`
}
```
